HTML

  • The body element represents the visible content shown to the user
  • An elemnty is refering to the entire element from openeing to closing

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is used to add styling to the context I created
  • With CSS I can add fonts, colors, spacing, etc..
  • CCS stands for Cascading style sheets

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git is a version control system that lets us manage and keep track of each version of our projects
  • Using git commnads we can initiate(or create a repo) add our code to the repo, and view different versions of our code

JavaScript
